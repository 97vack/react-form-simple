# Install

Just one command is all it takes to give `React-form-simple` a try.

Using npm:

```jsx | pure
npm install -S react-form-simple
```

Using yarn

```jsx | pure
yarn add -S react-form-simple
```

Using pnpm

```jsx | pure
pnpm add -S react-form-simple
```
