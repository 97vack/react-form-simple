# Install

Just one command is all it takes to give `React-form-simple` a try.

Using npm:

```jsx | pure
npm install react-form-simple -S
```

Using yarn

```jsx | pure
yarn add react-form-simple -S
```

Using pnpm

```jsx | pure
pnpm add react-form-simple -S
```
